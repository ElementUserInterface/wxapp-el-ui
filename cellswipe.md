# cellswipe

---

## 使用指南

在page.json中引入组件

```json
{
    "usingComponents": {
      "el-cellswipe": "component/_cellswipe/cellswipe"
    }
}
```

### 属性

| 名称 | 类型 | 默认 | 说明 | 版本 |
| :--- | :--- | :--- | :--- | :--- |
| color | String | "inherit" | 橡皮筋区域背景颜色，未设置时橡皮筋也不会生效 | 1.0.2 |
| show | String | '' | 初始显示区域，前提插槽存在，否则一样无效，默认状态下表示空字符串既可 | 1.0.2 |
